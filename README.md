# netflix-data-analysis
Data analysis project on Netflix dataset using Python (Pandas, NumPy, Matplotlib)

# Netflix Data Analysis

## Overview
This project performs exploratory data analysis (EDA) on a Netflix dataset to uncover trends in content distribution, production, and genres.

## Objectives
- Analyze the distribution of Movies vs TV Shows
- Explore content growth over time
- Identify top content-producing countries
- Analyze popular genres

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains information about Netflix titles, including:
- Type (Movie / TV Show)
- Country
- Date added
- Genre
- Duration

## Key Analysis

### 1. Movies vs TV Shows
Comparison of the number of movies and TV shows available on Netflix.

### 2. Content Growth Over Time
Analysis of how Netflix content has grown over the years.

### 3. Top Countries
Identification of the countries producing the most content.

### 4. Genre Analysis
Exploration of the most popular genres on Netflix.

## Results & Insights
- Movies are more common than TV Shows on Netflix
- Content increased significantly after 2015
- The United States produces the most content
- Drama and Comedy are among the most common genres
